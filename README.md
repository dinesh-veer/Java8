## Features Demonstrated

* **Lambda Expressions**
  Learn concise, inline function definitions replacing anonymous classes.

* **Functional Interfaces**
  Utilize single-method interfaces like `Consumer`, `Function`, `Predicate`, etc.

* **Streams API**
  Process collections in a functional style using operations like `map`, `filter`, `reduce`, and terminal actions such as `forEach`.

* **Method References**
  Simplify lambdas when calling existing methods (e.g., `ClassName::methodName`).

* **Optional (Optional<T>)**
  Avoid `NullPointerException` with optional values and functional handling.

* **Stream Collectors**
  Aggregate stream results using `collect()`, `Collectors.toList()`, `joining()`, etc.

* **Date/Time API (java.time)**
  Work with modern, immutable, and powerful date and time types (e.g., `LocalDate`, `LocalDateTime`, `Duration`).

## Sample Usage

1. **Clone the repository**:

   ```bash
   git clone https://github.com/dinesh-veer/Java8.git
   cd Java8
   ```

2. **Explore the examples**:

    * Browse the Java files under `src/` to see how each feature is implemented.

3. **Run the examples**:

    * Compile and execute the classes via your IDE or command-line tools like:

      ```bash
      javac src/**/*.java
      java YourExampleClass
      ```
